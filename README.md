# Electerm

A terminal emulator written in HTML, CSS, and Javascript with the Electron framework. It is supported on Windows, MacOS, and Linux.

## System Requirements:

- Windows - Windows 10 1809 or newer
- MacOS - MacOS Monterey or newer
- Linux - Ubuntu 22.04 or equivalent/newer
